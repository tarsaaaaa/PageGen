# 🎓 Assignment Front Page Generator

A simple yet powerful web app to generate clean, professional **front pages** instantly — directly in your browser.

No installs. No backend. Just fill, preview, and download 📄✨

---

## 🌐 Live Demo

 https://customfrontpagegenerator.netlify.app/

---

## ✨ Features

* 🧾 Generate professional assignment front pages
* ⚡ Real-time live preview
* 🎨 Clean and customizable layout
* 🔄 Dynamic fields (name, subject, submitted to/by, etc.)
* 🧠 Smart layout handling (long names auto-adjust)
* 🧩 Multiple templates (Lab Report / Lab Sheet)

---

## 🚀 How to use

1. Goto 👉 https://customfrontpagegenerator.netlify.app/
2. Fill in your assignment details
3. Choose the type (Lab Report / Lab Sheet)
4. Preview updates instantly
5. Download page as PDF
